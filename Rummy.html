About the Project

Moon Rummy is an engaging 13-card multiplayer game designed for Android and WebGL platforms. The game offers a seamless and competitive gaming experience, allowing players to enjoy the popular Rummy card game with friends or online opponents. Featuring intuitive controls, immersive graphics, and cross-platform compatibility, Moon Rummy ensures hours of strategic and entertaining gameplay.

Key Features

  • Supports multiplayer gameplay for Android and WebGL platforms
  • Immersive 13-card Rummy experience with smooth animations
  • Cross-platform play enabling seamless interaction between mobile and web users
  • Real-time matchmaking and leaderboards for competitive play

Technologies Used

  • Unity 3D for robust game development
  • SmartFoxServer (SFS) for real-time multiplayer networking
  • Android SDK and WebGL for cross-platform compatibility
  • C# scripting for implementing game mechanics and interactions
